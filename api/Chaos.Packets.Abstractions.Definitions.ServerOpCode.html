    Enum ServerOpCode

    OpCodes used when sending packets to a client

    Namespace: Chaos.Packets.Abstractions.Definitions
    Assembly: Chaos.Packets.Abstractions.dll
    Syntax
    public enum ServerOpCode : byte
    Remarks

    In networking, an opcode is used to identify the type of packet that is being sent. The opcode is generally part of a header near the beginning of the data stream
